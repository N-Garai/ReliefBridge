# ReliefBridge - Disaster Relief Platform

## Quick Start
1. pip install -r requirements.txt
2. cp .env.example .env  (then add your Appwrite creds)
3. python setup_appwrite.py
4. python app.py

## Features
- User Authentication
- Help Request System
- Real-Time Updates (Appwrite Realtime)
- GPS Geolocation
- Live Map (Leaflet)
- High Contrast UI
